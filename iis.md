# IIS란?

### 정의
- Internet Information Sevices의 약자
- MS Windows의 웹서버
- Microsoft에서 제공하는 웹 서비스를 하기 위한 도구
- Windows Server에 내장되어 무료로 사용할 수 있는 웹 서비스 도구
- 웹 사이트, 서비스 및 응용 프로그램을 안정적으로 호스팅하기 위한 안전하고 확장 가능하며 쉽게 관리할 수 있는 모듈식 플랫폼

### 기능
- IIS 관리자를 사용하여 IIS 기능을 구성하고 웹 사이트를 관리한다.
- FTP(파일 전송 프로토콜)를 사용하여 웹 사이트 소유자가 파일을 업로드 및 다운로드 할 수 있다.
- 웹 사이트 격리를 하기 때문에 서버의 다른 사이트를 방해하지 않는다.
- 클래식 ASP, ASP.NET, PHP 등의 다양한 기술을 사용하여 작성된 웹 응용 프로그램을 구성한다.
- Windows PowerShell을 사용하여 웹 서버에 대한 대부분의 관리 작업관리 과정을 자동화한다.
- 여러 웹 서버를 IIS로 관리할 수 있는 서버 팜으로 구성한다.
- NUMA 하드웨어을 최대한 활용하고, NUMA 사용 가능 서버의 성능을 최적화한다.
